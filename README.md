# Networking

Repositório de documentação técnica e scripts nas áreas de **redes, monitorização e infraestrutura**. Cada tema está organizado na sua própria pasta, para facilitar a navegação e a manutenção.

## Conteúdos

- **[Zabbix — Coleta de VLAN e Subnet](zabbix/)** — descoberta e monitorização de VLANs e subnets de equipamentos de rede via Zabbix, usando personalização: SNMP, Low-Level Discovery e scripts externos.
- **[net-inventory — Inventário de rede via SNMP](net-inventory/)** — ferramenta em Python que varre uma sub-rede, consulta os equipamentos por SNMP e gera um inventário em CSV e JSON.

## Segurança das informações

Este repositório **não deve conter** palavras-passe, *communities* SNMP reais, endereços IP internos sensíveis, ficheiros de configuração com segredos nem identificadores de equipamentos internos.

Utilize sempre dados fictícios, variáveis de ambiente e mecanismos próprios para a gestão de segredos.
